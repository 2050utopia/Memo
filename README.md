###用到的技术点：
 * MVP模式
 * 数据库采用了Realm
 * DataBinding,主要是在手势密码界面
 * 沉浸式状态栏
 * evenbus界面通信更新
 * 水波纹效果
 * 多聚道打包
 * 写了一句java 8 的lambda--(SplashActivity.java)
 
###基本功能

###v1.0.3
 * 直接手势滑动退出

###v1.0.0
 * 遵循Material Design
 * 密码保存
 * 密码查看
 * 密码分类
 * 主题切换
 * 手势解锁
 * 用户反馈

###后续更新：
