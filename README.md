# 怕死我的

####一款基于Google Material Design设计开发的Android密码管理客户端--怕死我的（PassWord）

###基本功能

###v1.0.0
 * 遵循Material Design
 * 密码保存
 * 密码查看
 * 主题切换
 * 手势解锁
