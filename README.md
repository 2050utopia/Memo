# 怕死我的
[apk下载体验](http://fir.im/bxr8?release_id=56b029adf2fc42739a000007)
####一款基于Google Material Design设计开发的Android密码管理客户端--怕死我的（PassWord）如果对你有帮助，欢迎多多的star

###基本功能
###v1.0.0
 * 遵循Material Design
 * 密码保存
 * 密码查看
 * 密码分类
 * 主题切换
 * 手势解锁
 * 用户反馈
